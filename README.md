
# Projek Web Alumni SDN 1 Sukamulya

## Deskripsi
Projek Web Alumni SDN 1 Sukamulya adalah aplikasi berbasis web yang dibuat untuk
mengelola dan menampilkan data alumni Sekolah Dasar Negeri 1 Sukamulya.
Aplikasi ini bertujuan mempermudah pendataan alumni serta penyampaian informasi
secara online dengan tampilan yang sederhana dan mudah digunakan.

## Tujuan Pembuatan
- Memudahkan pengelolaan data alumni sekolah
- Menyediakan media informasi berbasis web
- Sebagai pemenuhan tugas proyek mata kuliah

## Fitur Aplikasi
- Halaman beranda
- Informasi data alumni
- Halaman profil
- Halaman berita
- Halaman kontak

## Teknologi yang Digunakan
- HTML
- CSS
- JavaScript
- AppScript
